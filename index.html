Caption

Crescendo of hues / Scarlet flames dance and rejoice / No magenta here!

How is the artwork created?

T1. Create the SVG in Adobe Illustrator: 1.1. Create a new 800x800px file with default settings. 1.2. Use the polygon tool to put down a simple polygon (e.g. square). Use the pen tool to add anchor points on the edge of the polygon, and hold down Cmd to drag the points to the desired location. 1.2.1. Repeat this process, using polygons to block out shapes that compose the picture. 1.2.2. It is recommended to start with polygons with no fill and a bright-colored stroke for visibility. Once all of the polygons have been put down, their colors and strokes can be replaced appropriately. 1.2.3. Take care not to accidentally create curved paths using the pen tool. 1.3. Ensure that all polygons lie completely within the artboard. 1.4. Export the artboard using Export for Screens... 1.4.1. Select SVG as the format. 1.4.2. In Format Settings, make sure that Styling (under SVG) is set to "Presentation Attributes". 2. Once the SVG has been exported from Illustrator, further processing is necessary in a text editor: 2.1. Using regex, find (\s(\d+)(\.?)(\d*?))\s and replace with ,$1 (there is a space at the end of ,$1 ) 2.1.1. This inserts commas between points in a format expected by the rasterizer. 2.2. Using regex, find (\d+)(\.?)(\d*?)" fill= and replace with " fill= 2.2.1. This deletes the last number in the list of points for each polygon. If this number is not deleted, then the rasterizer will not fill in one of the triangles for each polygon. 2.2.2. If there is a rect element, its height may accidentally be deleted by this regex. To prevent this, use only polygons instead of rects, or replace the height value after running the find/replace. 2.3. In the svg tag, insert x="0px" y="0px" width="800px" height="800px" 3. Save. The SVG is now ready to be displayed by the rasterizer!

How long did it take?

5 hours
